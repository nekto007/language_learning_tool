Что такое модуль?

Модуль - это функциональный раздел приложения (например, учебная программа, книги, словарь).

Поля формы:
  • Код модуля - уникальный идентификатор для кода
  • Название - отображается в интерфейсе
  • Описание - дополнительная информация
  • Иконка - FontAwesome иконка для UI
  • Порядок - позиция в навигации
  • Blueprint - техническое имя Flask модуля
  • URL Prefix - префикс URL адресов
Настройки:
  • Активен - можно назначать пользователям
  • По умолчанию - выдается автоматически при регистрации
