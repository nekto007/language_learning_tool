# Remaining Lesson Types Schema Audit

Generated by: `scripts/audit_remaining_lesson_types.py`

Run against live DB to populate actual counts.
Run command: `python scripts/audit_remaining_lesson_types.py --output reports/remaining_types_audit.md`
Apply mode patches: `python scripts/audit_remaining_lesson_types.py --apply`

## Lesson types covered

| Type | Fields audited |
| --- | --- |
| pronunciation | items[].word, items[].phonetic (audio optional) |
| sentence_correction | mode field; incorrect_sentence in items or top-level |
| sentence_completion | items[].prompt, items[].answer (alternatives optional) |
| collocation_matching | pairs[].phrase, pairs[].translation |
| idiom | phrase, meaning_ru, example, example_ru |

## Notes

- sentence_correction: `--apply` patches missing `mode` based on CEFR level (A0/A1 → guided, A2/B1 → open, B2/C1/C2 → rubric)
- All other types are audit-only; gaps require manual content review
- Script supports `--type <name>` to audit a single lesson type
